# **App Name**: NeonVoyage Portfolio

## Core Features:

- Dynamic Frontend Architecture: Develops a fully responsive single-page application that ensures smooth performance across mobile, tablet, and desktop devices, leveraging lightweight CSS transforms and perspective for 3D effects without heavy engines.
- Cinematic Loading Experience: Implement a full-screen, dark loading animation with neon blue and purple glowing text ('ROHITH YP'), a thin scanning line effect, subtle particle background, and a glitch flicker before a smooth fade to the homepage.
- Hero & About Sections: Display core developer information in a hero section with blur-to-clear heading, subtitle slide-up, and a pulsing 'Explore My Work' button. The 'About' section clearly states the developer's use of 'manual coding expertise and AI-assisted productivity tools'.
- Interactive Skills & Projects Showcase: Showcase categorized skills in floating glass cards with 3D hover tilt and neon glow effects. Projects are displayed in interactive cards featuring website name, category, live preview links, and integrated neon-style pricing badges with 3D hover animations.
- Tiered Services Display: Present service offerings in three premium glass cards (Basic, Animated Modern, Premium 3D-Style) with detailed features and pricing, including gentle float animations and intensified glow/lift effects on hover.
- Integrated Contact & Footer: Provide clear contact information (phone, email) with click-to-call/email buttons and a modern glass contact form featuring fade-up, neon glow on focus, and button sweep animations, complemented by a simple footer with copyright and social placeholders.

## Style Guidelines:

- Primary Color (Neon Purple Accent): A vibrant, electric purple that captures a futuristic, high-tech feel. Hex: #821AFC (HSL: 250, 90%, 65%).
- Background Color (Deep Navy/Purple): A very dark, desaturated navy-purple providing a sophisticated, immersive 'control panel' atmosphere. Hex: #14121C (HSL: 250, 20%, 9%).
- Accent Color (Bright Sky Blue): An analogous, slightly less saturated bright blue for contrast and additional neon highlights. Hex: #9ECFFC (HSL: 220, 70%, 75%).
- Headline font: 'Poppins' (sans-serif) for a modern, geometric, and contemporary look. Body font: 'Inter' (sans-serif) for clean readability and objective styling, enhancing the modern tech aesthetic.
- Theme: Futuristic AI control panel with glassmorphism for UI elements, rounded corners for soft modernity, and premium spacing for a refined, expansive feel.
- Global Animations: All animations must be smooth and lightweight, utilizing CSS transforms for blur-to-clear, slide-up, pulse, parallax, 3D hover tilt/lift, and gentle floating effects, avoiding heavy 3D engines.
- Background Elements: Implement a subtle glowing grid at the bottom and floating particle animations for an enhanced futuristic backdrop. Mouse parallax tilt effect for dynamic interaction.
- Interactive Elements: Features like neon glows on hover/focus, button pulse, and sweep animations create a responsive and engaging user experience, adhering to a high-end, premium aesthetic.
